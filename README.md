## 它到底能干嘛？——比你想象的要全

先说清楚 EagleFiler 的定位。它不是一个简单的文件夹整理器，也不是笔记软件，它更像是你 Mac 上的一个"信息中转站+档案室"。

**一键导入，随手收集**

在浏览器、邮件客户端或 Finder 里，按一下 F1，当前页面或文件就直接导入到 EagleFiler 了。不需要拖窗口，不需要选保存路径，按键就完事。如果想导入时顺便打个标签加个备注，Option-F1 一步搞定。

这个流程对研究人员、学生、记者特别友好——你在看某篇论文、某条新闻、某封重要邮件，看到觉得有用，一个快捷键收进去，继续干别的，不打断节奏。

**搜索比 Spotlight 强**

很多人说 Spotlight 够用了，但 EagleFiler 的搜索有几个特别的地方：它不只搜文件名，支持搜 PDF 内容、邮件附件、标签、备注，甚至可以用布尔语法组合条件搜索。对于积累了大量文件的用户，这种"内容全文检索"的能力差距就很明显了。

**邮件管理几乎没有对手**

这是很多老用户反复提到的一点：EagleFiler 对邮件的支持是专门做过的。支持从 Apple Mail、Microsoft Outlook 导入邮件归档，存储格式是标准 mbox，保留完整附件和元数据，不会丢失任何东西。如果你有大量历史邮件需要归档备份，EagleFiler 在这块确实是独一档的选手。

**文件库可以加密**

AES 加密，连索引都一起加密，不只是给文件加个锁头。密码保护的文件库同样支持全文搜索，安全和易用两不误。

**数据不绑定，迁移方便**

EagleFiler 的库格式就是普通文件和文件夹，Finder 可以直接访问，Time Machine 也能正常备份。没有专有数据库，没有锁定风险，数据完全在自己手里。这一点对于在意数据主权的用户来说非常加分。
